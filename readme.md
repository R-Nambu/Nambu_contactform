# Contact form の課題

南部家のセブ島での支出（買いもの）一覧を記録する


## 入力フォーム index.html
 
### what 「品名」
何を買ったの？
==> フリーフォーム

### why 「購入理由」
何で買ったの？
==> 6項目のカテゴリーから選択
==> select スタイル使用

### who 「購入者」
誰が買ったの？
==> 3項目から選択
==> radio ボタン使用

### when 「日時」
いつ買ったの？
==> 「OK」ボタンクリック時の日時を自動で取得

### how much 「金額」
いくらだった？
==> 数字以外が入力されたらエラー表示したい
==> int型を指定？（未完成）

### comment 「特記事項」
特記事項は？
==> フリー入力
==> 特記事項の入力は任意なので、「OK」ボタン出現の要件から外す
