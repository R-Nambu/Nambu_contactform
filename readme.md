# Contact form の課題

南部家のセブ島での支出（買い物）一覧を記録する


## Features
 
### what 「品名」
何を買ったか、何に出費したか？
==> フリー入力

### why 「購入理由」
何で買ったの？
==> 5項目から選択
==> 

### who 「購入者」
誰が買ったの？
==> 3項目から選択
==> ラジオボタン使用

### when 「日時」
いつ買ったの？
==> 「OK」ボタンクリック時の日時を取得

### how much 「金額」
いくらだった？
==> int型を指定

### comment 「特記事項」
特記事項は？
==> フリー入力



## Note

・特記事項の入力は任意 「OK」ボタン出現の必要要件から外す
